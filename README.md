# 📊 Tableau Dashboard

## 📝 Overview

This project contains a Tableau dashboard used to visualize and analyze data.

## 🎯 Purpose

The purpose of this project is to convert raw data into meaningful insights using interactive visualizations. It helps in understanding patterns, trends, and performance.

## 🔍 Insights

* Identified key trends in the data
* Compared performance across categories
* Highlighted important metrics (KPIs)
* Helped in better decision-making

## 📁 Files

* `dashboard.twb` – Tableau dashboard file
* `dataset.csv` – Data used in the dashboard

## 🛠 Tools Used

* Tableau
* Excel / CSV

## 🚀 How to Use

1. Download the files
2. Open `dashboard.twb` in Tableau
3. Connect the dataset if needed
